Monthly Park Unique Attendances

This report will generate the unique attendances for a Park, per month between two dates. Made for Freeholds but could be any park. Can be copied to the clipboard after the report has run.
NOTE: You can choose to limit this report to local player attendance only by toggling the first checkbox below.
WARNING: This report can use a lot of data, do NOT run it on your cellular data plan.
