All Kingdom/Park Abbreviations

This report generates all the abbreviations for every park. These abbreviations can be used in most ORK fields that expect a username. So to narrow down search results you can either specify the kingdom abbreviation such as nb: for the Nine Blades, or further nb:ff to specify Felfrost. You have to have a space after the abbreviations like nb:ff kismet to look for kismet in Felfrost in the Nine Blades.
