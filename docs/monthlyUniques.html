Monthly Kingdom Unique Attendances

This report will generate the unique Kingdom attendances per Park, per month between two dates. The report also include unique signins across the Kingdom overall within a month. Can be copied to the clipboard after the report has run.
NOTE: You can choose to limit this report to local player attendance only by toggling the first checkbox below. For Parks this means only those from the park count, for the Kingdom overall, only if a player is from the Kingdom will they be in the uniques for that month. Otherwise, if you leave it unchecked, everyone counts for both Parks and the Kingdom.
WARNING: This report can use a lot of data, do NOT run it on your cellular data plan.
